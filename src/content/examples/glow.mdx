## GlowEffect

以前も使った <ExampleLink slug="effect" /> の 1 つである`GlowEffect`を使うと、コンポーネントを発光させることができます。

<DocLink document="https://docs.flame-engine.org/latest/flame/effects.html#gloweffect" />
<DocLink reference="https://pub.dev/documentation/flame/latest/effects/GlowEffect-class.html" />

なお、このエフェクトは以下の注意書きがあり、今後 API が変更される可能性があるようです。
ここではバージョン`1.8.1`で動作することを確認しています。

> This effect is currently experimental, and its API may change in the future.

## 描画する

`CircleComponent`に`GlowEffect`を追加しています。

`infinite`を設定することでエフェクトを繰り返します。
また、`reverseDuration`で逆再生する際の秒数を設定しています。

<ExampleCode file="glow/game.dart" />

これで発光表現ができました。
