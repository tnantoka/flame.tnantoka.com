## RectangleComponent

ゲーム内の要素は`Component`を継承したクラスで表現されます。
四角形を描画するには`RectangleComponent`を使います。
（`Component`の子の`PositionComponent`の子です）

<Diagram
  code={`
classDiagram
  Component <|-- PositionComponent
  PositionComponent <|-- RectangleComponent
`}
/>

`PositionComponent`は多くの要素のベースになるもので、
画面上に描画するのに必要な属性を持っています。

| 属性     | 概要                                                 |
| -------- | ---------------------------------------------------- |
| position | 座標                                                 |
| size     | 大きさ                                               |
| scale    | スケール                                             |
| angle    | 回転角度                                             |
| anchor   | 描画の基準位置。デフォルトは左上（`Anchor.topLeft`） |

`RectangleComponent`はこれらに加えて`paint`（デフォルトは白）を指定することで、任意の四角形を描画することができます。
